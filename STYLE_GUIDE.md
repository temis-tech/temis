# Руководство по стилям сайта "Радуга слов"

## Цветовая палитра

### Основные цвета
- **Градиент**: `linear-gradient(135deg, #667eea 0%, #764ba2 100%)`
  - Начальный: `#667eea` (фиолетово-синий)
  - Конечный: `#764ba2` (фиолетовый)

### Дополнительные цвета
- **Фон**: `#ffffff`, `#f8f9fa`
- **Текст**: `#333333`, `#666666`
- **Акцент**: `#667eea`
- **Успех**: `#28a745`
- **Футер**: `#2c3e50` → `#34495e`

## Типографика

### Заголовки
- **H1**: `2.8rem - 3rem`, `font-weight: 700`
- **H2**: `2.2rem - 2.8rem`, `font-weight: 700`
- **H3**: `1.5rem - 1.6rem`, `font-weight: 600`

### Текст
- **Основной**: `1rem - 1.2rem`, `line-height: 1.6 - 1.7`
- **Мелкий**: `0.9rem`, `color: #999`

## Компоненты

### Карточки
- `border-radius: 16px`
- `box-shadow: 0 4px 20px rgba(0, 0, 0, 0.08)`
- Hover эффект: `translateY(-8px)`, увеличенная тень
- Верхняя полоса градиента при hover

### Кнопки
- `border-radius: 50px` (полностью скругленные)
- Градиентный фон или белый с градиентным текстом
- Hover: `translateY(-3px)`, увеличенная тень

### Анимации
- `transition: all 0.3s ease`
- Fade in up для появления элементов
- Плавные переходы для всех интерактивных элементов

## Адаптивность

### Breakpoints
- **Desktop**: `> 768px`
- **Mobile**: `≤ 768px`

### Мобильные изменения
- Уменьшенные отступы (`1.5rem` вместо `2rem`)
- Одна колонка вместо сетки
- Упрощенная навигация (бургер-меню)
- Уменьшенные размеры шрифтов

## Эффекты

### Градиенты
- Используются для заголовков (text gradient)
- Фоновые градиенты для hero секций
- Акцентные полосы на карточках

### Тени
- Легкие тени для карточек: `0 4px 20px rgba(0, 0, 0, 0.08)`
- Усиленные при hover: `0 8px 30px rgba(102, 126, 234, 0.2)`

### Hover эффекты
- Подъем элементов: `translateY(-5px to -8px)`
- Изменение цвета текста/фона
- Увеличение тени
- Масштабирование изображений: `scale(1.1)`
